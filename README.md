# THE ELEMENTS LITE

Simple resource pack I use that modifies a few textures. The first version of this pack was made back in 2020 and was a near-total conversion of the 1.12 base game textures.
However, this pack, which is to be constantly maintained, is a distillation of the original down to some of the most (subjectively) important textures and resources.
